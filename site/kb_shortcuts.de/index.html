    Tastaturkürzel

    Dieses Blatt beschreibt viele nützliche Tastenkombinationen für ODIN. Einige dieser Tastaturkürzel entsprechen den Kurzbefehlen anderer Programme (zum Beispiel Strg+Cbzw.Cmd+Cfür das Kopieren).

    Microsoft Windows

    Allgemein
    F11 Vollbild ein/aus
    Alt Strg+M Projektverwaltung öffnen
    Alt Strg+B Basiskartenverwaltung öffnen
    Anzeige
    Strg+Shift++ Zoom hinein (vergrößern)
    Strg+- Zoom heraus (verkleinern)
    Strg+0 Zoom normal
    Bearbeiten
    Klick Objekt auswählen
    Alt+Klick Punkt löschen
    Umsch+Klick markiertes (Mehrpunkt-)Objekt verschieben
    Umsch+Klick Bereich auf Karte mit Maus aufziehen
    STRG+Klick mehrere Objekte auf der Karte auswählen (Bereich)
    Strg+A Alles auswählen
    Strg+C Kopieren (Zwischenablage)
    Strg+V Einfügen (Zwischenablage)
    Strg+X Ausschneiden (Zwischenablage)
    Entf Objekt löschen
    Strg+Z Rückgängig
    Strg+Schift+Z Wiederherstellen
    Verschiedenes
    Strg+M Fenster mimmieren
    Strg+W Anwendung schließen

    Mac OSX

    Allgemein
    ControlCommandF Vollbild ein/aus
    Command+F Suche öffnen
    Command+P Palette öffnen
    Anzeige
    Command++ Zoom hinein (vergrößern)
    Command+- Zoom herraus (verkleinern)
    Command+0 Zoom normal
    Command+L Zeige Beschriftung (bis 1:250.000)
    Command+1 Kartenwerk 1
    Command+(n) Kartenwerk (n)
    Bearbeiten
    Klick Element auswählen
    Ctrl+Klick Punkt löschen
    Command+A Alles auswahlen
    Command+C Kopieren (Zwischenablage)
    Command+V Einfügen (Zwischenablage)
    Command+X Ausschneiden (Zwischenablage)
    Command+Rück Element löschen
    Ctrl+Rück Element löschen
    Command+Z Rückgängig
    Command+Schift+Z Wiederherstellen
    Verschiedenes
    Command+B Lesezeichen anlegen
    Option+C Koordinate kopieren (Zwischenablage)
    Command+H Anwendung ausblenden
    Command+W Fenster schließen
    CommandQ Anwendung schließen
